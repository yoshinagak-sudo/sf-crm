# sf-crm

Early prototype work that informed the design of [butaifarm-crm](https://butaifarm-crm.vercel.app/) — Butai Farm's in-house Salesforce replacement.

The production CRM (now live, with 587 accounts and 870 contacts migrated off Salesforce) lives in a private repository. This repo holds early structural experiments around account-contact modeling and activity timelines.

## Stack

- Next.js / TypeScript
